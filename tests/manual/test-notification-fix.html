通知组件修复测试

测试1: 堆叠通知显示修复
测试修复后的堆叠通知功能，特别是当通知内容超过三行时的显示效果。
修复内容：动态计算通知高度，避免通知重叠问题。
测试用例：
- 短文本通知（1-2行）
- 中等长度通知（3-4行）
- 长文本通知（5行以上）
- 混合长度通知堆叠
测试2: 阴影效果增强
测试增强后的阴影效果，提升通知的视觉层次感。
修复内容：将阴影从 0 10px 15px -3px rgba(0, 0, 0, 0.1) 增强为 0 20px 25px -5px rgba(0, 0, 0, 0.15)
测试3: 综合功能测试
测试通知组件的完整功能，包括不同类型、位置和配置选项。
测试说明
修复内容总结：
1. 堆叠通知修复：动态计算通知高度，解决长文本通知重叠问题
2. 阴影效果增强：提升阴影深度，增强视觉层次感
3. 高度计算优化：添加估算逻辑，确保通知元素未完全渲染时也能正确计算高度
