介绍 FCL

FCL (Fold Craft Launcher) 是一个为移动设备设计的 Minecraft 启动器，让你可以在手机或平板上畅玩 Minecraft Java 版。

它具有以下特点：

  • 移动端优化: 专为触屏设备设计，操作流畅。
  • 多架构支持: 支持多种 CPU 架构，兼容性好。
  • 社区驱动: 由活跃的社区开发和维护。
  • 开源: 代码公开，安全可靠。

更多信息请访问 FCL GitHub 仓库。
